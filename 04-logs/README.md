# Event Logs

## Tools

- RegRipper

## What I did
- reviewed login events;
- checked system and security logs;
- searched for suspicious actions;
- analyzed timestamps and user activity;
- compared events with the incident timeline.

## Result
- found important log entries;
- identified traces of suspicious activity.

## Conclusion
Learned the basics of event log analysis.
